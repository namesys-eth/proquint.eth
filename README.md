# Proquint.eth

Proquints are Readable, Spellable, and Pronounceable Identifiers. 
- Range : bytes4 (0x00000000 ~ 0xffffffff)
- Total Supply : 4,294,967,295
- Mint : 1 ID per Address

### Formats : 

- Subdomain : `<pq1>`.`<pq2>`.proquint.eth 
- Subdomain : `<pq1>`-`<pq2>`.proquint.eth 
- Subdomain : `<pq1><pq2>`.proquint.eth 
- App: `<pq1>`-`<pq2>`@proquint.eth 
- Standalone : pq://`<pq1>`.`<pq2>`

### Example :

- 0x00000000 = babab.babab.proquint.eth
- 0xffffffff = zuzuz-zuzuz.proquint.eth
- 0x4db42fdf = hukuh.zizuf.proquint.eth
